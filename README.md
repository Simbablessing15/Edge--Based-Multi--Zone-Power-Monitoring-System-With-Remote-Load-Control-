Edge-Based Multi-Zone Power Monitoring System with Remote Load Control

This project presents an ESP32-based smart energy management system designed to monitor and control electrical loads across multiple zones in real time. The system integrates PZEM-004T power meters for energy monitoring, PIR sensors for occupancy detection, relay modules for load switching, and Firebase for remote monitoring and control.

The solution employs an edge-based load prioritization algorithm that automatically manages connected loads based on occupancy status and power consumption. By making decisions locally on the ESP32, the system minimizes latency, reduces dependence on cloud services, and maintains operation even during network interruptions.

Key features include real-time power monitoring, occupancy-aware load control, remote dashboard access, automated load shedding, and multi-zone energy management. The system is designed to improve energy efficiency, reduce operational costs, and provide a scalable solution for smart buildings, educational institutions, and commercial facilities.
